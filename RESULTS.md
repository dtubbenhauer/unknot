All counts below are relative to the input snapshot recorded in `data/`.

## Lower-bound improvements

The corrected and completed searches produce 422 distinct improvements: 372
exact unknotting numbers and 50 non-exact lower-bound improvements.

### Finite-field Alexander module

For a Seifert matrix `V`, finite field `F`, and nonzero `alpha` in `F`, the
calculation records

```text
nullity_F(alpha V - V^T) <= e(K) <= u(K).
```

The finite-field calculation is exhaustive. If `E_r` is generated by the
`(m-r+1)`-minors of `tV-V^T`, then the largest finite-field nullity is

```text
beta(K) = max { r : E_r is not the unit ideal }.
```

The complete elementary-ideal scan checks all 5,121 database rows with a gap.
The index distribution is `beta=0,1,2,3` for respectively
`11, 4504, 588, 18` rows. It produces 363 successful lower-bound certificates;
one is the already-recorded `12n873` consistency case, leaving 362 genuinely
new improvements:

- 303 exact values `u=2`;
- 14 exact values `u=3`;
- 42 improvements `[1,3] -> [2,3]`;
- two improvements `[1,4] -> [2,4]`; and
- one improvement `[2,5] -> [3,5]`.

The completion adds thirteen certificates to the earlier restricted scan:
eleven exact values `u=2` and two improvements `[1,3] -> [2,3]`. Eight use
`F_16`, four use `F_49`, and one uses `F_81`. The exact names and witnesses are
in `lower_bounds/alexander_module/additional_certificates.csv`.

For a nullity-`r` witness from an `m`-square Seifert presentation, an extension
degree at most `m-r+1` always suffices. Only finitely many primes differ from
the characteristic-zero behavior for a fixed presentation, and that finite set
is effectively computable. There is no genus-only bound on those primes. The
finite-field index satisfies `beta(K) <= e(K) <= beta(K)+1`; the possible gap is
real, so the exact Nakanishi index requires one final global generator test.
